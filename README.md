# ⚙️ How To Start

```shell
// 시작 전
api key 발급 및 https 환경 구성이 필요합니다. 개발 환경 설정 시 발생하는 문제점은 채널톡을 통해 문의주세요

// 처음 프로젝트를 실행할 경우
yarn install

// 프로젝트 시작
yarn start:https

// 개발 페이지 접속
https://local.gatherplz.com:3000/
```
